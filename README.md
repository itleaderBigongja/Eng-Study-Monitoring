# 영어 학습 플랫폼 & 모니터링 시스템

## 프로젝트 구조
- eng-study: 영어 학습 백엔드 (Spring Boot)
- eng-study-frontend: 영어 학습 프론트엔드 (Next.js)
- study-monitoring: 모니터링 백엔드 (Spring Boot)
- study-monitoring-frontend: 모니터링 프론트엔드 (Next.js)

## 개발 환경
- Java 17
- Node.js 22
- PostgreSQL 16
- Elasticsearch 8.11
- Kubernetes

## 시작하기
1. Ubuntu 서버 준비
2. `./deploy.sh` 실행
3. http://[Ubuntu-IP]:3000 접속
